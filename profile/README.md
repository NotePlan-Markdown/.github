# NotePlan — Notes + Tasks + Calendar in Unified Markdown Workspace

## About NotePlan

NotePlan combines daily notes, task management, and calendar events in a single Markdown-based workspace. NotePlan links notes across days, weeks, and projects using bi-directional backlinks. NotePlan integrates directly with system calendars (Google, iCloud, Exchange) to show events alongside your tasks. NotePlan enables users to time-block their day, write meeting notes, and review weekly progress — all in plain text Markdown. NotePlan supports tags, filters, search, and custom templates. NotePlan is trusted by productivity enthusiasts, project managers, writers, entrepreneurs, and knowledge workers worldwide.
